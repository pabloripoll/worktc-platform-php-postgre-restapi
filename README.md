# WORKTIME CONTROLLER - SYMFONY 7

Este repositorio contiene un ejemplo básico de un servicio API RESTful desarrollado con **Symfony 7**, diseñado para fines de investigación y como demostración de mi perfil de desarrollador. Implementa las características principales de una aplicación back-end minimalista y personalizada de control de tiempo de trabajo y sirve como proyecto de referencia para aprendizaje, experimentación o como ejemplo de código de desarrollo back-end.

> ⚠️ **Project Status: In Development**
>
> Este repositorio se encuentra actualmente en desarrollo y aún no está listo para su uso. Las características y API pueden cambiar, y pueden producirse cambios importantes. Tenga en cuenta que el código base está en constante evolución.
